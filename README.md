# CTR_Prediction_ML
Predicting ad clicks with Machine Learning, Python, and Data Mining Techniques.

## Project Purpose
The primary purpose of this project is to apply machine learning techniques to predict Click-Through Rates (CTR) for programmatic AdOps (Advertising Operations). By accurately forecasting which users are most likely to engage with an ad, this model helps optimize real-time bidding strategies and maximize campaign performance. 

This project features an interactive **Dashboard Model** built to visualize campaign data and allow AdOps managers to run live bid predictions.

## Dataset
The predictive model was trained and evaluated using a robust ad engagement dataset sourced from **Kaggle**, containing user demographics, internet usage behavior, and historical click data.
Dataset Link: https://www.kaggle.com/datasets/swekerr/click-through-rate-prediction?resource=download
